# Installation
Install corvu with the package manager of your choice:

<Code code={`
npm install corvu
`} lang="bash" copyButton />

You're good to go! Head over to [Usage](/docs/state/) to learn how to use corvu.

## TailwindCSS plugin
If you want make use of the tailwind modifiers like `corvu-open:`, install the tailwind plugin:

<Code code={`
npm install @corvu/tailwind
`} lang="bash" copyButton />

Then add the plugin to your `tailwind.config.js` file:

<Code code={`
module.exports = {
  // ...
  plugins: [
    // Use it with the default prefix 'corvu'
    require('@corvu/tailwind'),
    // or with a custom prefix
    require('@corvu/tailwind')({ prefix: 'ui' }),
    // ...
  ],
}
`} lang="js" />

Check out the [TailwindCSS](/docs/styling/#tailwindcss-plugin) section in the styling guide to learn more about the tailwind plugin.
